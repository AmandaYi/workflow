* `vue_init_h5_web_vw` 这一套配置项目适合的是移动端开发,移动端适配VM,面向广大web用户,不过也适用了pc,看以下说明
--- 使用场景
vw与vh会在pc与移动端均产生效果，而不像flexible只会转换一定宽度（记得是750px）以下设备的px为rem，因此若设计稿为移动端而生，全权使用vw单位会使得页面在pc端出现字体过大等现象，需酌情处理，根据应用场景自行选择
